# polishit.skill 🪣✨

> **polish** (抛光) + **shit** (屎) = 给屎抛光
>
> 帮打工人把形式主义的"屎"，雕成领导想要的"花"。

---

## 这是什么

一个给 AI Agent 用的 Skill。

装上它，你的 AI 助手就变成了一个精通体制内外各种形式主义文书的"公文精灵"——你只需要告诉它大概发生了什么，它负责帮你把废话堆满，把成果吹大，把检讨写得诚恳又无害。

**它不帮老板压榨你，它帮你对付老板布置的作业。**

---

## 能干什么

| 模块 | 适用场景 | 触发词 |
|------|----------|--------|
| 通讯稿生成器 | 执法动态、工作简报、被扣工资的那种 | 通讯稿、新闻稿、工作动态 |
| 周报/月报膨胀器 | 明明没干啥，要写满字数 | 周报、月报、日报 |
| PPT大纲生成器 | 三句话的事要整15页PPT | PPT、汇报、述职 |
| 年终总结转化器 | 干了一年杂活，要冲击评优 | 年终总结、述职报告 |
| 整改报告生成器 | 被检查组点名，要"举一反三" | 整改报告、立行立改 |
| 民主生活会生成器 | 批评与自我批评，要说又不能真说 | 民主生活会、对照检查 |
| 领导讲话稿填充器 | 代写领导讲话，提高站位狠抓落实 | 讲话稿、发言稿 |

## 安装方法

### WorkBuddy / OpenClaw / Claude Code

```bash
git clone https://github.com/YOUR_USERNAME/polishit.skill ~/.workbuddy/skills/polishit
```

或者直接把 `SKILL.md` 文件下载到你的 skills 目录下。

### 手动安装

把 `SKILL.md` 放到你的 AI 工具的 skills 文件夹，重启即可。

## 使用示例

**你说：**
> 帮我写个通讯稿，这周在春季行动里查了几家无证烟草户

**polishit 输出：**
> 【聚焦春季·严查无证】XX局持续深化春季市场整治行动
>
> 近日，XX局结合春季专项整治行动部署，聚焦无证经营这一市场顽疾，持续加大执法力度……（400字成品，可直接提交）

---

**你说：**
> 周报，这周开了两个会，跟进了A客户，其他时间在熟悉新系统

**polishit 输出：**
> 一、深度跟进A客户需求，完成关键决策人触达与需求画像更新，为下一阶段商务推进奠定基础。
> 二、组织召开项目复盘会与需求评审会，对齐多方认知，明确执行路径与风险预案。
> 三、完成新系统操作流程学习与实操演练，提升个人工作效率，缩短任务响应周期。

## 设计理念

```
不是帮你卷赢同事，而是帮你把被浪费的时间夺回来。
```

那些设计形式主义制度的人，赌的就是你会把时间耗在这些事情上——耗到你没精力思考、没时间成长、没心气反抗。

而 polishit.skill 的逻辑是：**用你教我的规则，用最短的时间把你布置的作业写完，然后转身去做我真正想做的事。**

这不是消极怠工，这是战略性应付。

---

- 陪家人吃一顿完整的晚饭
- 读完那本买了一年没拆封的书
- 什么都不做，只是发呆
