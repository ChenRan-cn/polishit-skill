# 示例：整改报告生成器

## 示例一：台账不规范整改

**输入：**
```
检查组说我们台账不规范，部分记录缺失
```

**输出：**

**关于XX检查组反馈问题的整改报告**

接到检查组反馈意见后，我单位党政领导高度重视，第一时间召开专题会议，深刻剖析问题根源，明确责任分工，制定整改方案，现将整改落实情况报告如下：

**一、提高站位，端正态度。**充分认识台账管理是规范化建设和依法行政的基础性工作，对检查组反馈的问题诚恳接受、照单全收，坚决杜绝"就事论事"的片面认识，切实把整改工作摆在重要位置。

**二、立行立改，逐一销号。**一是组织专人对现有台账进行全面系统梳理，逐项核查，补录缺失记录，确保台账信息完整准确；二是建立"日清周结"工作机制，明确台账更新时限要求，做到当日事当日录；三是明确专人负责台账日常管理，将台账规范化管理纳入岗位职责考核，以考核促规范。

**三、举一反三，建章立制。**以此次问题整改为契机，对照检查组意见，对同类工作进行全面"回头看"，修订完善《XX工作台账管理规范》，从制度层面堵塞管理漏洞，建立健全长效机制，确保台账管理规范化、常态化。

下一步，我单位将持续巩固整改成效，建立整改"回头看"制度，防止问题反弹，确保各项工作经得起检验。

## 示例二：工作作风问题整改

**输入：**
```
被反映工作作风不扎实，服务意识不强
```

**输出：**

**关于工作作风问题的整改报告**

对照检查反馈意见，本人深感责任重大、警醒深刻。现就工作作风方面存在的问题，认真查摆、深刻剖析，诚恳接受批评，并制定以下整改措施：

**一、深刻认识，提高站位。**认真学习习近平总书记关于作风建设的重要论述，深刻认识到良好的工作作风是做好一切工作的根本保障。对照党员干部标准，深挖思想根源，真正在思想上来一次彻底的洗礼。

**二、立即整改，补齐短板。**一是切实增强服务意识，把群众的事当自己的事来办，做到有呼必应、有问必答；二是进一步提升工作标准，凡事力求精益求精，杜绝粗枝大叶、草率了事；三是主动走访联系群众，深入一线了解实际情况，切实解决群众急难愁盼问题。

**三、建立机制，长效巩固。**建立定期自查机制，对照党纪党规和岗位职责，每月开展一次作风自查，及时发现和纠正存在的问题，确保作风建设取得实实在在的成效。

以此次整改为镜，本人将以更加饱满的工作热情、更加严格的工作标准，全力以赴做好各项工作，以实际行动回应组织和群众的关切与期望。
