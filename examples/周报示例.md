# 示例：周报/月报膨胀器

## 示例一：普通打工人周报

**输入：**
```
这周主要跟进了一下A客户，然后内部开了两个会，其他时间在熟悉新系统，还有就是帮同事看了一下方案
```

**输出：**

**本周工作总结**

一、深度跟进A客户需求，完成关键决策人触达与需求画像更新，就下阶段合作方向达成初步共识，为后续商务推进奠定坚实基础。

二、组织参与项目复盘会与需求评审会，联动多部门对齐工作进展与阶段目标，明确各方执行路径与风险预案，有效提升跨部门协同效率。

三、完成新业务系统操作流程系统性学习与实操演练，深入掌握核心功能模块，持续提升个人工作效率，缩短任务响应周期。

四、协助同事开展方案评审工作，结合业务实际提出优化建议，助力团队整体工作质量提升。

**下周计划**

持续推进A客户跟进工作，深化系统操作熟练度，配合团队完成各项既定工作目标。

## 示例二：月报膨胀版

**输入：**
```
这个月就是常规工作，开了几个会，处理了一些日常事务，没啥特别的
```

**输出：**

**XX月工作总结**

一、扎实推进日常业务工作。坚持系统思维，持续优化工作流程，严格落实各项规范要求，确保日常工作高效有序运转，有效保障了各项业务指标的平稳推进。

二、积极参与各类会议研讨。认真贯彻落实会议精神，深入领会工作要求，及时将会议部署转化为具体工作举措，确保各项任务按时按质完成。

三、高质量完成日常事务性工作。严格按照工作标准，认真处理每一项工作任务，注重细节把控与质量提升，做到事事有落实、件件有回音。

**下月计划**

持续巩固日常业务工作成效，进一步提升工作效率，配合团队完成各项既定目标任务。
