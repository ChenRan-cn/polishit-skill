# 示例：通讯稿 / 工作动态生成器

## 示例一：市场调研通讯稿

**输入：**
```
本周部门搞了一次用户需求调研，覆盖30家企业，收集了15条反馈意见
```

**输出：**

【深耕客户·服务升级】XX部门扎实推进用户需求专项调研工作

近日，XX部门紧紧围绕"以客户为中心"服务理念，聚焦用户真实需求，组织开展用户需求专项调研活动，全面摸排服务现状，持续提升服务质量。

本次调研采取"问卷+走访"相结合的方式，累计覆盖目标企业30家。通过面对面交流、电话访谈、线上问卷等多渠道收集意见建议，共收集各类反馈意见15条。调研内容涵盖产品使用体验、服务响应速度、个性化需求等关键维度。

下一步，XX部门将深入分析调研数据，对收集到的意见建议进行分类整理，形成问题清单，明确改进方向，持续优化产品和服务，不断提升用户满意度。

## 示例二：安全生产检查通讯稿

**输入：**
```
社区开展安全生产检查，走访了20家商户，发现3处隐患已整改
```

**输出：**

【安全生产·常抓不懈】XX社区深入推进安全生产专项检查工作

为进一步筑牢安全生产防线，近日，XX社区组织工作人员深入辖区开展安全生产专项检查行动，全面排查各类安全隐患，切实保障辖区居民群众生命财产安全。

本次行动采取"走访+督查"相结合的工作模式，累计走访辖区商户20家，重点检查消防设施配备、电气线路使用、安全出口设置等关键环节。对现场发现的3处安全隐患，工作人员当场下达整改通知，督促经营者立行立改，目前3处隐患已全部完成整改销号。

下一步，XX社区将持续加大安全生产检查力度，建立"日常检查+集中整治"常态化工作机制，推动安全生产责任落实落细，不断提升辖区安全生产管理水平。

## 示例三：产品上线通讯稿

**输入：**
```
公司新功能上线了，叫智能推荐，用户转化率提升了12%
```

**输出：**

【产品升级】XX平台智能推荐功能正式上线，转化率提升12%

为持续优化用户体验，近日，XX平台"智能推荐"功能正式上线运营。该功能基于用户行为数据分析，通过算法模型精准匹配用户需求，有效提升信息获取效率和内容相关性。

上线以来，平台核心指标持续向好。数据显示，新功能上线后，用户转化率较上线前提升12%，用户停留时长平均增加约20%，整体使用体验得到显著改善。

下一步，团队将持续迭代优化算法模型，引入更多维度特征标签，不断提升推荐精准度，为用户带来更加个性化的使用体验。
